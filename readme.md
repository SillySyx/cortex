<div align="center">
    <h1><code>Cortex</code></h1>
    <strong>An offline password manager and knowledgebase application.</strong>
</div>


# Features
* Password manager
* Knowledgebase
* Data encryption
* Webassembly
* Offline progressive web application


# Requirements
* Rust

Setup
```
cargo install wasm-pack
cargo +nightly install miniserve
```
